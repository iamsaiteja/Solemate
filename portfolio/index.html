01 / ABOUT

Code that ships

I'm a software engineer from Hyderabad who takes projects the whole way — from data model to REST API to pixel-level frontend polish to the server it runs on. By day I work as a Data Migration Analyst at Aazad Engineering, moving and validating enterprise data with Python and SQL; by night I build and ship full-stack products.

My flagship project, SoleMate, is a multi-seller e-commerce platform with a scroll-driven WebGL 3D hero, Razorpay UPI/card payments, Google OAuth, JWT auth and a hand-rolled glassmorphism design system — Django 6 + DRF on AWS EC2, React 19 on Vercel, zero animation libraries.

I care about performance budgets, clean APIs and the small details that make an app feel finished.
